# B-Fit — Android Fitness App

A fitness guide Android application built as my CS50 final project, 
designed to help beginners get started on their fitness journey.

## Demo
[Watch Video Demo](https://youtu.be/Fur4ETcbrQE)

## About
B-Fit serves as a guide for anyone who doesn't know where or how to 
get started with fitness. The app covers workouts, nutrition, and the 
mindset necessary for self-improvement.

## Features
- **Workout Plans** — Beginner (4-day), Intermediate (5-day), and Advanced (6-day) split programs
- **Nutrition Guide** — Covers the fundamentals of nutrition and daily food breakdown
- **Macro Calculator** — Calculates daily caloric intake based on height, weight and age
- **About Me** — Personal fitness journey and direct contact via TikTok

## Tech Stack
- Java
- Android Studio
- XML (UI Layout)

## How to Run
1. Clone the repo
2. Open in Android Studio
3. Run on an emulator or physical Android device

- Android Activity navigation
